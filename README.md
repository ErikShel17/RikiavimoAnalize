# 📊 Rikiavimo algoritmų analizė

*Projektas atliktas studijų tikslais.*

## 🧠 Aprašymas
Šiame projekte lyginami du rikiavimo algoritmai:

- **Insertion Sort** – paprastas, efektyvus mažiems arba beveik surikiuotiems duomenims (O(n²))
- **Merge Sort** – stabilus ir greitas algoritmas dideliems duomenims (O(n log n))

Tikslas – praktiškai įvertinti jų veikimo skirtumus.

## ⚙️ Funkcionalumas
- Duomenų nuskaitymas iš failų  
- Rikiavimo laiko matavimas  
- Palyginimų ir sukeitimų skaičiavimas  
- Testavimas su skirtingais duomenų rinkiniais:
  - nesurikiuoti
  - atvirkščiai surikiuoti
  - jau surikiuoti

## 🛠️ Reikalavimai
- C++11 arba naujesnis
- Visual Studio (.sln projektas) arba kita C++ aplinka
